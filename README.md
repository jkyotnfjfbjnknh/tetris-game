# 俄罗斯方块 (Tetris Game)

一个使用原生HTML、CSS和JavaScript实现的经典俄罗斯方块游戏。

## 功能特性

- 经典俄罗斯方块玩法
- 实时分数、等级和消除行数统计
- 预览下一个方块
- 响应式设计，支持移动端
- 键盘控制

## 操作说明

- ← → : 左右移动方块
- ↑ : 旋转方块
- ↓ : 加速下降
- 空格 : 瞬间下落
- P : 暂停/继续游戏

## 游戏规则

- 每消除1行获得100分 × 当前等级
- 消除2行获得300分 × 当前等级
- 消除3行获得500分 × 当前等级
- 消除4行获得800分 × 当前等级
- 每消除10行提升1个等级，游戏速度加快

## 技术实现

- 使用HTML5 Canvas进行渲染
- 原生JavaScript实现游戏逻辑
- CSS3实现视觉效果和响应式布局

## 如何运行

只需打开 `index.html` 文件即可开始游戏。
